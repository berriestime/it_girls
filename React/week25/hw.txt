1.  Для начала работы с ReactJS нужно установить node и npm;
2.  Плюсы SPA:
    - Быстрое создание;
    - Все, что нужно для SPA - JS;
    - Производительность приложения;
    - Обработка большого количества данных;
    Минусы SPA:
    - Не работает семантика;
    - Необходимость интернет соединения;
    - Обязательна поддержка JS;
3.  Создать React-приложение можно командой
    create-react-app firstapp
4.  React это библиотека, но можно назвать и фреймворном. Похожий функционал есть у Vue и Angular
5.  Добавить библиотеку можно командой
    mpn install lib
6.  formik, react bootstrap, material-UI, blueprint
7.  App.js - основной файл
8.  Папка build нужна для хранения файлов которые будут запущены в другом окружении.
    Создается командой npm build
